# School App Privacy Policy Page

This folder contains a minimal Next.js website for the School App privacy policy.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Notes

- The page content is based on the current `SchoolApp` mobile codebase.
- Update the contact section before publishing.
- If you want, this page can later be expanded with a dedicated `/privacy-policy` route, branding, logo, or Hindi version.
